# ssgit

自作の git クローン (サブセット)

## Installation
```sh
cargo install -f --git https://github.com/SSlime-s/ssgit
```

## Usage
```
Usage: ssgit [COMMAND]

Commands:
  cat-file      
  hash-object   
  update-index  
  write-tree    
  ls-files      
  commit-tree   
  update-ref    
  init          
  add           
  commit        
  switch        
  branch        
  help          Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
